Команда для запуска : "java -jar sort.jar", далее пишем аргументы в любом порядке.
Первый идентификатор, не являющийся ключом, будет воспринят, как имя выходного файла.
Файлы для проверки должны находиться в директории sort.
Запуск происходит из директории sort.
Использованная команда для компиляции : "jar cfvm sort.jar MANIFEST.MF -C classes ."(Из директории sort).
